# 전국 자동차 검사소 표준데이터

'자동차관리법'에 따라 국토교육부 장관이 지정한 자동차검사소 정보, <br>
전국에 있는 자동차 검사소의 위치와, 검사기술 인력 수, 자동차의 어떤부분을 점검하는지 <br>
matplotlib, seaborn로 시각화를 해 봅니다.

* 결측값 대체
  * 숫자 0으로 대체 합니다.
* 소수점 제거를 위해 int로 형 변환
* '소재지번주소' 컬럼으로 지역명을 추출 합니다.
